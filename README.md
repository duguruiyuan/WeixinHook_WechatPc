# WeixinHook
c#微信机器人 PC hook 稳定不掉线 微信个人号接口 （自动通过、拉群，踢人，发图片，自动回复）<br><br><br>

目前可实现功能：<br>
1、根据微信号、wxid、昵称发送文本消息。<br>
2、根据wxid发送图片消息。<br>
3、获取群信息<br>
4、群活群成员列表<br>
5、群内发送消息<br>
6、删除群成员<br>
7、自动通过好友<br>
...........
<br><br>
非web网站接口，是基于pc版本微信制作。<br>
目前hook版本的微信放出来的极少，很多都有限制。这个版本我已经亲自应用到项目当中，相当稳定。<br>
